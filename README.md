> NATO Phonetic Alphabet

Based on NATO gem by Lukas Alexandre (https://github.com/lukelex/NATO)

## Installation

```sh
$ npm install --save nato
```

## Usage

```js
var Nato = require('nato');

var text = new Nato('test');

text.nato
// Tango Echo Sierra Tango
```
